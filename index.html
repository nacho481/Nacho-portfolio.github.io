üì¶ Mailman Delivery System (TSP + Hash Table Optimization)

Designed a robust CLI-based logistics simulator applying the Traveling Salesman Problem (TSP) and custom data structures to streamline delivery routes. This project combines algorithmic efficiency with modular OOP and terminal-based visualization.

It features a custom Hash Table with Linear Probing for real-time package tracking, leveraging a 2D distance matrix to compute delivery routes. The Nearest Neighbor Algorithm is used to solve TSP efficiently. This project avoids linked-list chaining to prevent potential O(n) time complexity from long collision chains, favoring linear probing for its localized resolution strategy.
